# Hello World

Welcome to tiago-ai — a blog/dump of AI conversations in the life of a dev/RSE interested in bioimaging, wiki, linked data, iNaturalist and plants.

## How to add a post

Drop any `.md` file into the `posts/` directory and restart the server. That's it.

```bash
node server.js
```

## About

This blog is intentionally minimal. No build step, no CMS, no database. Just markdown files.
